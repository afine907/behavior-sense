# 前端控制台交互设计规范

> BehaviorSense Admin Console - 详细交互设计与组件规范

---

## 一、设计原则

### 1.1 核心原则

| 原则 | 说明 |
|------|------|
| 简洁高效 | 减少点击次数，关键操作一目了然 |
| 实时反馈 | 操作立即响应，状态清晰可见 |
| 容错设计 | 危险操作需确认，支持撤销 |
| 一致性 | 相似功能交互模式统一 |

### 1.2 布局规范

```
┌──────────────────────────────────────────────────────────────┐
│  Header (64px)                                               │
│  ┌─────────┬────────────────────────────┬──────────────────┐ │
│  │ Logo    │        Breadcrumb          │  User Avatar     │ │
│  └─────────┴────────────────────────────┴──────────────────┘ │
├────────────┬─────────────────────────────────────────────────┤
│            │                                                 │
│  Sidebar   │                 Main Content                    │
│  (240px    │                 (min-height: calc(100vh-64px))  │
│  collapsed:│                                                 │
│  64px)     │   ┌─────────────────────────────────────────┐   │
│            │   │  Page Header (Title + Actions)          │   │
│            │   ├─────────────────────────────────────────┤   │
│            │   │                                         │   │
│            │   │  Content Area                           │   │
│            │   │  (padding: 24px)                        │   │
│            │   │                                         │   │
│            │   └─────────────────────────────────────────┘   │
│            │                                                 │
└────────────┴─────────────────────────────────────────────────┘
```

### 1.3 通用组件

| 组件 | 用途 | 规范 |
|------|------|------|
| PageHeader | 页面标题 + 操作按钮 | 标题 h2，右侧 Primary/Secondary 按钮 |
| DataTable | 数据列表 | 搜索、筛选、分页、排序 |
| DetailCard | 详情卡片 | 标题 + 内容，可折叠 |
| EmptyState | 空状态 | 图标 + 说明 + 操作引导 |
| LoadingState | 加载状态 | Skeleton 骨架屏 |
| ConfirmDialog | 确认弹窗 | 标题 + 说明 + 取消/确认 |

       │
       ▼

       │
       ▼

│ 方式2: 高级模式 (AST 编辑器)   │

       │
       ▼
配置动作

│ Action Types:                  │

       │
       ▼

       ▼

       │
       ▼

       │
